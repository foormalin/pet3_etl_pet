# pet3_etl_pet

## Описание

Данный проект представляет собой простой пакетный ETL-пайплайн, разработанный с использованием Python, PostgreSQL, Docker и Apache Airflow.

В рамках проекта реализованы основные этапы обработки данных:

* извлечение данных из CSV-файла;
* очистка и преобразование данных;
* загрузка данных в PostgreSQL;
* создание аналитической витрины данных с помощью SQL.

## Используемые технологии

* Python
* pandas
* PostgreSQL
* SQLAlchemy
* Apache Airflow
* Docker
* Docker Compose

## Структура проекта

```text
etl/        — логика ETL-процесса
dags/       — DAG для Apache Airflow
sql/        — SQL-скрипты
data/       — исходные и обработанные данные
tests/      — модульные тесты
```

## Запуск проекта

```bash
docker compose up --build
```

После запуска контейнеров ETL-пайплайн можно выполнить через веб-интерфейс Apache Airflow.

## Возможности проекта

* пакетная обработка данных (Batch ETL);
* очистка и преобразование данных;
* загрузка данных в PostgreSQL;
* оркестрация пайплайна с помощью Apache Airflow;
* контейнеризация с использованием Docker;
* базовое тестирование компонентов ETL.
